# 帕秋莉任务手册

1.15.2的 BQ 和 FTBQ 都暂时没有出来，我就趁机做了一个帕秋莉任务手册。

## 优点

- 数据驱动，版本控制友好，只需更改 json 文件即可修改任务~~（复制友好）~~。
- 支持多本任务手册。
- 比 BQ 更加好看。
- 与帕秋莉手册兼容，更加容易书写教程。
